## Commands
1. cd ~/.ssh
   ssh-keygen -> use ```mmgkey``` as a name
2. terraform plan
3. terraform apply
4. ```ssh -i ~/.ssh/mmgkey ubuntu@<public-IP-address-of-the-EC2-instance>```
5. terraform plan -destroy
6. terraform apply -destroy
